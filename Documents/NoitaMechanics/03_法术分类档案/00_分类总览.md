# Noita 法术分类总览

> 生成日期：2026-06-07。数据来自 `gun_actions.lua` 与 `common.csv`，共解析 422 个 action。

## 类型统计

| 原始类型 | 中文分类 | 数量 | 主要底层作用 |
|---|---:|---:|---|
| `ACTION_TYPE_MODIFIER` | 投射修正法术 | 143 | 不一定自己生成投射物，主要修改当前 ConfigGunActionInfo 状态 c，影响后续投射物。 |
| `ACTION_TYPE_PROJECTILE` | 投射物法术 | 122 | 执行后通常生成移动 ProjectileComponent 实体，或生成带触发器的投射物。 |
| `ACTION_TYPE_STATIC_PROJECTILE` | 静态投射物法术 | 45 | 生成场、区域、云、光束、十字等更偏固定或持续区域效果的实体。 |
| `ACTION_TYPE_OTHER` | 其他_特殊法术 | 42 | 执行随机、复制、刷牌、希腊字母、终结类等特殊逻辑。 |
| `ACTION_TYPE_MATERIAL` | 材料法术 | 26 | 生成或转换材料，通常用 c.material 或特定实体/脚本实现。 |
| `ACTION_TYPE_UTILITY` | 实用法术 | 25 | 偏功能性移动、传送、工具、召唤或状态效果。 |
| `ACTION_TYPE_DRAW_MANY` | 多重释放法术 | 14 | 通过 draw_actions(n, ...) 或类似逻辑额外抽取多张法术卡。 |
| `ACTION_TYPE_PASSIVE` | 被动法术 | 5 | 作为法杖持有/始终存在效果，常在库存或手持时生效。 |

## 字段说明

- `mana`：卡牌 action 被抽取执行时从法杖当前法力扣除的值；未声明时游戏使用默认值。
- `max_uses=-1`：数据中通常表示无限使用；部分无限法术在 Lua 中用注释保留 `max_uses=-1`。
- `spawn_level` / `spawn_probability`：随机法杖/商店等生成池中的层级和权重，逗号一一对应。
- `related_projectiles`：卡牌 UI 与反射系统关联的实体 XML；真正生成哪个实体仍以 `action()` 中的 `add_projectile*` 调用为准。
- `action_summary`：由脚本自动抽取的行为摘要，只用于快速定位；复杂法术仍需查看原始 `action()`。摘要生成时已剔除 Lua 行注释。
