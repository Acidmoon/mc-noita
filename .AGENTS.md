# MC Noita Project Instructions

## Target Platform

- Minecraft Java Edition: `1.20.4`
- Mod loader: Fabric
- Primary language: Java
- Java runtime: JDK 17, preferably Eclipse Temurin / Adoptium
- IDE: IntelliJ IDEA Community with the Minecraft Development plugin

## Core Build Stack

- Use Gradle Wrapper from the project root. Prefer `./gradlew` or `.\gradlew.bat`; do not rely on a globally installed Gradle.
- Use Fabric Loom for Minecraft decompilation, mappings, run configurations, and remapped JAR output.
- Use Yarn mappings for examples and implementation unless the project is explicitly migrated.
- Build and publish only the remapped JAR produced by Loom, not the development JAR.

Recommended baseline versions:

```properties
minecraft_version=1.20.4
yarn_mappings=1.20.4+build.3
loader_version=0.15.11
loom_version=1.6-SNAPSHOT
fabric_version=0.97.1+1.20.4
```

`fabric_version=0.92.0+1.20.4` is the older documented baseline. This workspace uses `0.97.1+1.20.4` to match the local Fabric 1.20.4 game instance.

## Fabric Dependencies

Minimum expected dependencies:

```groovy
dependencies {
    minecraft "com.mojang:minecraft:${project.minecraft_version}"
    mappings "net.fabricmc:yarn:${project.yarn_mappings}:v2"
    modImplementation "net.fabricmc:fabric-loader:${project.loader_version}"
    modImplementation "net.fabricmc.fabric-api:fabric-api:${project.fabric_version}"
}
```

Optional dependencies should be added only when needed:

- Cloth Config v12 for in-game configuration screens.
- Mod Menu v8 for Fabric mod list integration.
- Custom Portal API only if custom portals are required.

## Development Defaults

- Main entrypoint: `ModInitializer`
- Client entrypoint: `ClientModInitializer`
- Data generation entrypoint: `DataGeneratorEntrypoint`
- Keep shared logic in common code and client-only rendering/input/UI code in client-only entrypoints or source sets.
- Do not reference `net.minecraft.client.*` from common server-loadable code.
- Prefer Fabric API events before introducing Mixins.
- When Mixins are needed, prefer `@Inject`, accessor/invoker Mixins, or MixinExtras operations. Avoid `@Overwrite`.
- Split common and client Mixin configuration files when targeting client-only classes.

## Minecraft 1.20.4 Specifics

- Use Java 17. Do not upgrade the project runtime to Java 21 unless the Minecraft target changes to 1.20.5+.
- Data generation APIs differ from 1.21+. For recipes in 1.20.4, use `generateRecipes(Consumer<RecipeJsonProvider>)`.
- Networking uses the pre-1.20.5 Fabric networking API:
  - `ClientPlayNetworking.send(id, buf)`
  - `ServerPlayNetworking.registerGlobalReceiver(...)`
  - `ServerPlayNetworking.send(player, id, buf)`
  - `ClientPlayNetworking.registerGlobalReceiver(...)`
- Do not use the 1.20.5+ Payload networking style for this project.
- GUI and screen handler registration should follow the 1.20.4 Fabric API patterns, not 1.21+ examples.

## Gradle Tasks

Common tasks:

```powershell
.\gradlew.bat genSources
.\gradlew.bat runClient
.\gradlew.bat runServer
.\gradlew.bat runDatagen
.\gradlew.bat build
```

Use `runClient` for local gameplay testing, `runServer` for dedicated-server compatibility checks, and `runDatagen` for generated JSON assets.

## Resource And Data Conventions

- Mod IDs and resource paths must be lowercase. Prefer short names with underscores.
- Keep generated data under the configured datagen output path and review generated JSON before committing.
- Register blocks, items, item groups, block entities, entities, events, and networking handlers during the correct Fabric initialization phase.
- Prefer data-driven JSON for recipes, loot tables, tags, models, worldgen definitions, and language files when practical.

## Release Guidance

- Use `.\gradlew.bat build`.
- Publish the remapped release JAR from `build/libs/`.
- Do not publish `*-dev.jar`.
- Target release platforms: Modrinth, CurseForge, and optionally GitHub Releases.
- In `fabric.mod.json`, keep version constraints aligned with the stack:

```json
"depends": {
  "fabricloader": ">=0.15.11",
  "minecraft": "~1.20.4",
  "java": ">=17",
  "fabric-api": "*"
}
```
