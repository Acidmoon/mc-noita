## 法术属性说明

| 属性 | 说明 |
|------|------|
| 法力消耗 | 施放该法术消耗的法力值 |
| 施法延迟 | 对每次施法冷却时间的修正（负值=更快） |
| 使用次数 | 法术耗尽前可被施放的次数 |
| 投射物伤害 | 投射物命中目标造成的直接伤害 |
| 爆炸伤害 | 爆炸时造成的范围伤害 |
| 冰霜伤害 | 冰霜属性伤害 |
| 闪电伤害 | 闪电属性伤害 |
| 切割伤害 | 切割类伤害（用于锯片等） |
| 速度 | 投射物初始飞行速度 |
| 散射 | 投射物发射方向的随机偏差 |
| 持续时间 | 投射物存在时间（帧数） |
| 爆炸半径 | 爆炸影响的圆形区域半径 |
| 暴击率 | 投射物命中时暴击的几率 |
| 定时 | 定时触发的时间（帧数） |
